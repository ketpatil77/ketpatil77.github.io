# Portfolio App

Vite + React portfolio app for Ketan Patil.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

From the workspace root you can also use:

```bash
npm run install:app
npm run dev
npm run build
```

## Content

- App source: `src/`
- Static assets served by Vite: `public/`
- Site content and section data: `src/config.ts`
